Detailed Description

Abstract class for use with Load Sharing Facility (LSF) job management tools.

Subclasses must be override the run method to handle implementation- specific IPC. run methods should check that self.numActiveJobs never exceeds self.max_active_jobs.

Subclasses may override default implementation for self.jobCompleteCallback() if they wish to perform an action when a job is marked as complete.
